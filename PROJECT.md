# Planet Hosts

A self-hosted WHM/cPanel-style web hosting panel with integrated streaming engine, game server management, and billing system.

## Features

- **Hosting Management** — Create/manage hosting accounts, packages, DNS, email, FTP, MySQL, SSL certificates
- **Streaming Engine** — Built-in radio streaming via Icecast/SHOUTcast with AutoDJ, listener stats, and per-stream management
- **Game Servers** — One-click game server deployment and management
- **Billing System** — Products, orders, invoices, payments, coupons, taxes, credits, refunds
- **Reseller System** — Tiered reseller accounts with permissions and account assignment
- **Backup & Restore** — Full and per-user backups with profiles, history, and restore preview
- **Migration Wizard** — Import from cPanel, Plesk, DirectAdmin, SonicPanel, and Centova Cast
- **Terminal** — Browser-based SSH terminal with persistent working directory
- **Dashboard Widgets** — Customizable dashboard with movable, resizable widgets
- **Security** — Admin permissions, super admin protection, IP blocking, firewall controls

## Tech Stack

- **Backend**: PHP 8.2, custom MVC framework
- **Database**: MariaDB/MySQL
- **Frontend**: Vanilla JS, CSS with dark theme
- **Server**: Apache with PHP-FPM
- **Streaming**: Icecast, SHOUTcast DNAS (v1/v2)
